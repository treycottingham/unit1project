Feb 27 2018

Meetup Information

Web App Security Meetup

People I Met

Erich Hartnett - Head of Recruitment - SecureSet; Serge Borso - SpiderSec CEO

What I Learned

I learned how to access hidden files within the browser. I did not realize that it was so easy (and free) to download software that allows for penetration testing. We used a software called Burp to intercept the communication between the server and the browser of the person accessing the data. We used this tool as well as metadata to find out such things as location, salary info, and social security numbers for employees in a fake company whose files were poorly hidden.

Questions I Have

How can our program lead to a career in CyberSecurity. What languages are used in penetration testing or how much fluency must you have in different languages in order to be a decent hacker?
